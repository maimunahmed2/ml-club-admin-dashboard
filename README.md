## Overview

**ML Club Admin Dashboard** is a clean and responsive admin interface designed to simplify the internal management of a Machine Learning club.

Instead of manually updating club data across different places, this dashboard brings everything into one organized space. Admins can manage members, projects, blogs, events, announcements, and other important club content through a smooth and modern web interface.

The project focuses on practical dashboard workflows, clean UI design, reusable components, and a professional admin experience.

## Features

- Modern and responsive admin dashboard
- Clean sidebar-based navigation
- Dashboard overview for club activity
- Member management interface
- Project management section
- Blog and content management section
- Event and announcement management
- Reusable UI components
- Mobile-friendly layout
- Fast and smooth user experience
- Clean visual design suitable for real club operations

## Why I Built This

ML clubs often manage a lot of information, including members, projects, blogs, events, and announcements. As the club grows, handling everything manually can become messy and time-consuming.

This project was built to solve that problem by creating a centralized dashboard where club admins can manage important content in a simple and organized way.

It also helped me improve my skills in building real-world admin dashboards, designing better user interfaces, and structuring frontend projects professionally.

## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/maimunahmed2/ml-club-admin-dashboard.git
```

### 2. Move into the project folder

```bash
cd ml-club-admin-dashboard
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open the project in your browser

```txt
http://localhost:5173
```
